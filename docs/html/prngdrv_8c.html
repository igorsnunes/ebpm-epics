Detailed Description

Driver support for BPM. Controls hardware parameters using ZeroMQ.

Author
Igor Nunes
Date
14 Mar 2014 This Driver Support provides access to hardware using an API for accessing and modifing its parameters and acquiring data. The API is build over ZeroMq.

int bpmConfig ( const char *  portNumber)

Function for initial configuration of Driver Support called by st.cmd.

*This function exports standard interfaces for use in device support, register a port for communicating with hardware and setting some basic hardware configuration. *For each interface, some functions are defined for: connecting/disconecting driver support, *reading/writting values from lower layers.
